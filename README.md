# Github Finder

A clean and professional web application to search for GitHub users and view their profiles. Built using HTML5, CSS3, and modern JavaScript (ES6).

## Features

- Search for GitHub users by username
- View user profile details including avatar, bio, repositories, followers, and more
- Responsive and modern UI design
- Fast and intuitive user experience

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Start searching for GitHub users!

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
